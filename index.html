Triangle Tracker

Description of application

This application will help the user know if three values entered for lengths of a triangle produce an Isosceles, Equilateral, or Scalene triangle, or no triangle at all.

Directions for use

Kindly follow these instructions to use the application:

  1. Fill the form below with three integers that you think would produce a triangle
  2. Press the submit button
  3. View result for the type of triangle

Triangle Tracker Form

Fill the form below with three integers (whole numbers) that represent lengths of a triangle
